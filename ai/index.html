🤖 Автоматизация и агенты

ИИ-агенты

ИИ-агент — чатик с возможностью выполнять действия (отправка писем, создание таблиц). Автоматизация естественным языком.

  • Manus [интжвр] — универсальный ИИ-агент
  • n8n — визуальный редактор ИИ-агентов
  • Composio — галерея MCP-инструментов

Фреймворки и платформы

  • LangChain — Python-фреймворк для создания ИИ-агентов
  • Pipedream — low-code платформа для создания workflow и triggered serverless
    • Пример workflow: создание эндпоинта, который получает название покемона в query-параметре → обращается к Pokemon API → передает результат ChatGPT для интерпретации → сохраняет в БД + генерирует картинки + отправляет в Slack
    • Поддерживает MCP: можно вызывать workflow прямо из чата с ChatGPT

Интеграции

📚 Обучение и справка

Терминология

  • RAG — Retrieval Augmented Generation — поиск и передача кастомного контекста модели из векторной БД
  • MCP — Model Context Protocol — продвинутый API для передачи контекста ИИ

Полезные материалы

  • 📝 RAG (Retrieval Augmented Generation) — простое и понятное объяснение — понятная статья на Хабре
  • 📹 N8N всё. Escalator AI всё делает сам, и сразу ;) — сравнение инструментов
  • 📹 ИИ-агенты — вот что действительно изменит разработку. Пишем ИИ-агент на Python, LangChain и GigaChat — практическое руководство с LangChain и GigaChat
