ИИ-агенты

Чатик с возможностью выполнять действия (отправка писем, построение табличек). Иными словами, автоматизация естественным языком.

  • manus [интжвр] - дженерал ии-агент
  • n8n - визуальный редактор ии-агентов
  • composio - галлерея mcp

MCP - Model Context Protocol - апишка на стероидах для скармливания ии

  • langchain - python-фреймворк для декларации функций, которые будет дергать ии-агент

Терминология

  • RAG — Retrieval Augmented Generation — поиск и передача кастомного контекста модельки из векторной бд

Материалы / Видосы

  • 📝 RAG (Retrieval Augmented Generation) — простое и понятное объяснение
  • 📹 N8N всё. Escalator AI всё делает сам, и сразу ;)
  • 📹 ИИ-агенты — вот что действительно изменит разработку. Пишем ИИ-агент на Python, LangChain и GigaChat
