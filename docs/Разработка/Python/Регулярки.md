# Регулярки

## Как сделать повтор символа?

`r'\d{8}'` - число повторяется 8 раз

## Как сделать матчинг любого символа в тч \n?

Исользовать `.` в сочетации с `re.DOTALL`:

```python
re.findall(
    r'^M:(.*?)T:(.*?)D:(.*?)W:(.*?)$',
    commit_msg,
    re.DOTALL,
)
```
